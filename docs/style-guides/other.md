# Misc. Language Style / Usage

- CSS should be linted with `npm run lint:css`
- HTML should be formatted with `npx prettier --check '**/*.html'` and issues can be fixed automatically with `npx prettier --write '**/*.html'`
- Github actions should be formatted with `actionlint`
- Jinja2 template files should be named `<name>.html.jinja2` (e.g. `catan_lobby.html.jinja2`)
